# 📂 Auto List The Files and Folders

A simple and efficient Python tool to display all files and folders inside a given directory path.  
This project helps users quickly explore directory contents with clear labeling of files and folders.

---

## 🚀 Features

- 📁 Lists all folders in a directory  
- 📄 Lists all files in a directory  
- ✅ Validates if the given path exists  
- ❌ Handles invalid paths safely  
- ⚡ Fast and lightweight  
- 💻 Clean terminal output  
- 🧑‍💻 Beginner-friendly  

---

## 🛠️ Requirements

- Python 3.x  
- No external libraries required (uses built-in `os` module)

---

## ▶️ How to Run

1. Clone the repository

git clone https://github.com/rsamwilson2323-cloud/auto-list-the-files-and-folders.git

2. Navigate to the project folder

cd auto-list-the-files-and-folders

3. Run the script

python "auto list the files and folders.py"

---

## 💻 How It Works

- User enters a folder path  
- Program checks if the path exists  
- Displays all contents inside the folder  
- Differentiates between files and folders
